# Глоссарий

**🟢 Уровень 1: Начинающий**

Словарь терминов glm и математики для графики (C++/Vulkan).

| Термин                                           | Объяснение                                                                                                                                                                                                                                                                       |
|--------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **glm**                                          | OpenGL Mathematics — header-only библиотека, повторяющая типы и функции GLSL в C++. Векторы, матрицы, кватернионы, трансформации.                                                                                                                                                |
| **GLSL**                                         | OpenGL Shading Language. Язык шейдеров; glm следует его именованию и соглашениям (например, column-major матрицы).                                                                                                                                                               |
| **vec2, vec3, vec4**                             | Векторы из 2, 3 и 4 компонент (float). В игре: vec3 — позиция, направление; vec2 — UV, размер; vec4 — цвет с альфой, однородная координата.                                                                                                                                      |
| **mat3, mat4**                                   | Квадратные матрицы 3×3 и 4×4. В игре: преобразование позиции вершины — матрица модели (где объект), вида (камера), проекции (на экран).                                                                                                                                          |
| **quat**                                         | Кватернион — представление поворота без гримлока (gimbal lock). Тип `glm::quat`, а также `eulerAngles`, `slerp`, `mat4_cast` — в **gtc/quaternion.hpp** (стабильно). Дополнительные функции (`toMat4`, `rotate(quat, vec3)`, `squad`) — в **gtx/quaternion.hpp** (experimental). |
| **column-major**                                 | Порядок хранения матрицы по столбцам. В glm и Vulkan/GLSL матрицы column-major; первый индекс — столбец.                                                                                                                                                                         |
| **radians**                                      | Углы в радианах. В GLSL и glm функции вроде `rotate` принимают угол в радианах; `glm::radians(degrees)` переводит из градусов.                                                                                                                                                   |
| **glm::radians**                                 | Переводит угол из градусов в радианы. Нужен для `glm::rotate`, `glm::perspective` (FOV) и т.п.                                                                                                                                                                                   |
| **glm::translate**                               | Строит матрицу переноса (translation). Использование: `glm::translate(mat4(1.0f), vec3(x, y, z))`.                                                                                                                                                                               |
| **glm::rotate**                                  | Строит матрицу поворота вокруг оси на угол (в радианах).                                                                                                                                                                                                                         |
| **glm::scale**                                   | Строит матрицу масштабирования по осям.                                                                                                                                                                                                                                          |
| **glm::perspective**                             | Строит матрицу перспективной проекции (FOV в радианах, aspect, near, far).                                                                                                                                                                                                       |
| **glm::ortho**                                   | Строит матрицу ортогональной проекции (лево, право, низ, верх, near, far).                                                                                                                                                                                                       |
| **glm::lookAt**                                  | Строит матрицу вида камеры: позиция камеры, точка взгляда, вектор «вверх».                                                                                                                                                                                                       |
| **glm::value_ptr**                               | Возвращает указатель на непрерывную память вектора или матрицы. Нужен для записи в uniform-буфер Vulkan или `vkCmdPushConstants`. Из `<glm/gtc/type_ptr.hpp>`.                                                                                                                   |
| **glm::mat4_cast**                               | Преобразует кватернион в матрицу поворота 4×4. Из `gtc/quaternion.hpp`.                                                                                                                                                                                                          |
| **glm::quat_cast**                               | Преобразует матрицу поворота 3×3 или 4×4 в кватернион. Из `gtc/quaternion.hpp`.                                                                                                                                                                                                  |
| **glm::make_vec3**, **make_mat4**, **make_quat** | Строят glm-тип из указателя на непрерывную память. Используются для чтения из отображённой памяти Vulkan (uniform, буфер). Из `<glm/gtc/type_ptr.hpp>`.                                                                                                                          |
| **EXT**                                          | Стабильные расширения glm (ext/*): не входят в ядро GLSL, но полностью протестированы. Например, `ext/matrix_transform.hpp`, `ext/matrix_clip_space.hpp`.                                                                                                                        |
| **GTC**                                          | Рекомендуемые расширения glm (gtc/*): стабильные, с тестами. Например, `gtc/matrix_transform`, `gtc/type_ptr`, `gtc/quaternion`.                                                                                                                                                 |
| **GTX**                                          | Экспериментальные расширения glm (gtx/*). Требуют `#define GLM_ENABLE_EXPERIMENTAL` и включения конкретного заголовка (например, `gtx/quaternion.hpp`). API может меняться.                                                                                                      |
| **GLM_FORCE_LEFT_HANDED**                        | Макрос: использовать левую систему координат (по умолчанию — правая). Определять до первого include glm.                                                                                                                                                                         |
| **GLM_FORCE_DEPTH_ZERO_TO_ONE**                  | Макрос: использовать диапазон глубины [0, 1] (Vulkan) вместо [-1, 1] (OpenGL).                                                                                                                                                                                                   |
| **GLM_FORCE_SWIZZLE**                            | Макрос: включить swizzle-операторы (`.xyz`, `.rgb`, `.stp`). Значительно увеличивает время компиляции.                                                                                                                                                                           |
| **GLM_FORCE_ALIGNED_GENTYPES**                   | Макрос: использовать выровненные типы (требуется для SIMD оптимизаций).                                                                                                                                                                                                          |
| **GLM_FORCE_SIMD_AVX2**                          | Макрос: принудительно использовать инструкции AVX2 (требует поддержки CPU).                                                                                                                                                                                                      |
| **GLM_FORCE_EXPLICIT_CTOR**                      | Макрос: запретить неявные преобразования между типами (например, `ivec4 → vec4`).                                                                                                                                                                                                |
| **GLM_FORCE_SIZE_T_LENGTH**                      | Макрос: заставить `.length()` возвращать `size_t` вместо `int`.                                                                                                                                                                                                                  |
| **GLM_ENABLE_EXPERIMENTAL**                      | Макрос: разрешить включение экспериментальных расширений GTX.                                                                                                                                                                                                                    |
| **uniform, uniform-буфер**                       | Данные, передаваемые в шейдер на каждый кадр или объект (например, матрица модели). В C++ записываются в буфер Vulkan через `value_ptr`; шейдер читает их как uniform.                                                                                                           |
| **NDC**                                          | Normalized Device Coordinates — координаты вершины после применения проекции. По осям X, Y обычно в диапазоне [-1, 1]; по Z — [-1, 1] (OpenGL) или [0, 1] (Vulkan). Задаётся макросом `GLM_FORCE_DEPTH_ZERO_TO_ONE`.                                                             |
| **vec1, dvec3, ivec2, uvec4, bvec3**             | Специализированные векторные типы: `vec1` (1 компонент), `dvec3` (double), `ivec2` (int), `uvec4` (unsigned), `bvec3` (bool).                                                                                                                                                    |
| **glm::mix**                                     | Линейная интерполяция: `mix(a, b, t) = a + t*(b-a)`. Работает со скалярами, векторами, матрицами.                                                                                                                                                                                |
| **glm::clamp**                                   | Ограничение значения между min и max: `clamp(x, min, max)`.                                                                                                                                                                                                                      |
| **glm::step, glm::smoothstep**                   | Ступенчатая функция и её сглаженный вариант (используются в шейдерах).                                                                                                                                                                                                           |
| **glm::transpose**                               | Транспонирование матрицы.                                                                                                                                                                                                                                                        |
| **glm::determinant**                             | Определитель матрицы.                                                                                                                                                                                                                                                            |
| **glm::inverse**                                 | Обратная матрица.                                                                                                                                                                                                                                                                |
| **glm::dot, glm::cross**                         | Скалярное и векторное произведения.                                                                                                                                                                                                                                              |
| **glm::normalize**                               | Нормализация вектора (приведение к длине 1).                                                                                                                                                                                                                                     |
| **glm::reflect, glm::refract**                   | Отражение и преломление векторов (для освещения).                                                                                                                                                                                                                                |
