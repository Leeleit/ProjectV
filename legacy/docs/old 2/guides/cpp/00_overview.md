# C++ Handbook для ProjectV

**Полное руководство по Modern C++** для разработки воксельного движка.

C++ — основной язык разработки ProjectV, объединяющий производительность низкоуровневого системного программирования
с выразительностью современных стандартов (C++20/23/26). Это руководство охватывает все аспекты C++ в контексте игрового
движка — от базовых концепций до продвинутых паттернов DOD/ECS.

## Требования

### Обязательные

- **Компилятор C++ с поддержкой C++20** (MSVC 2019+, GCC 11+, Clang 12+)
- **Базовое понимание C++** (классы, шаблоны, STL)
- **CMake 3.20+** для сборки примеров

### Рекомендуемые инструменты

- **CLion/VS Code** с поддержкой C++20
- **clang-tidy и clang-format** для статического анализа
- **Tracy** для профилирования производительности
