# Обзор GLM

🟢 **Уровень 1: Начинающий**

**GLM (OpenGL Mathematics)** — header-only математическая библиотека для графики на C++. Повторяет типы и функции OpenGL
Shading Language (GLSL): векторы, матрицы, кватернионы. Предназначена для расчёта трансформаций, камеры и освещения в
графических приложениях.

Версия: **1.0.0+** (поддержка C++17 и выше)
Исходники: [g-truc/glm](https://github.com/g-truc/glm)
Документация: [manual.md](https://github.com/g-truc/glm/blob/master/manual.md)

## Основные возможности

- **GLSL-совместимые типы** — vec2/3/4, mat2/3/4, кватернионы
- **Функции трансформации** — translate, rotate, scale, perspective, lookAt
- **Математические функции** — dot, cross, normalize, inverse, determinant
- **Header-only** — не требует компиляции, только include
- **Без внешних зависимостей** — только стандартная библиотека C++
- **Кроссплатформенность** — Windows, Linux, macOS

## Структура библиотеки

```
glm/
├── core/       # Ядро: GLSL-подобные типы и функции
├── ext/        # Стабильные расширения
├── gtc/        # Рекомендуемые расширения (стабильные)
└── gtx/        # Экспериментальные расширения (требуют GLM_ENABLE_EXPERIMENTAL)
```

### Приоритет использования

1. **core** — базовые типы (vec, mat) и функции
2. **gtc** — стабильные расширения (matrix_transform, quaternion, type_ptr)
3. **ext** — альтернативные заголовки для оптимизации компиляции
4. **gtx** — экспериментальные функции (API может меняться)

## Карта заголовков

| Категория         | Заголовки                                            | Назначение                                           |
|-------------------|------------------------------------------------------|------------------------------------------------------|
| **Векторы**       | `<glm/vec2.hpp>`, `<glm/vec3.hpp>`, `<glm/vec4.hpp>` | vec2, vec3, vec4 (float, int, uint, bool, double)    |
| **Матрицы**       | `<glm/mat3x3.hpp>`, `<glm/mat4x4.hpp>`               | mat3, mat4                                           |
| **Трансформации** | `<glm/gtc/matrix_transform.hpp>`                     | translate, rotate, scale, perspective, lookAt, ortho |
| **Кватернионы**   | `<glm/gtc/quaternion.hpp>`                           | quat, slerp, angleAxis, mat4_cast                    |
| **Указатели**     | `<glm/gtc/type_ptr.hpp>`                             | value_ptr, make_mat4, make_vec3                      |
| **Константы**     | `<glm/gtc/constants.hpp>`                            | pi, epsilon, infinity                                |
| **Глобальный**    | `<glm/glm.hpp>`                                      | Всё ядро + базовые расширения                        |

## Требования

- C++17 или новее (для версии 1.0.0+)
- Без внешних зависимостей (только стандартная библиотека)
- CMake 3.10+ (рекомендуется)
