# Dear ImGui

**🟢 Уровень 1: Начинающий**

**Dear ImGui** — библиотека графического интерфейса для C++. Используется для создания инструментов отладки, редакторов
и игровых UI. Отличается отсутствием сохраненного состояния виджетов (immediate mode): интерфейс создается и
отрисовывается каждый кадр заново.

Исходники: [ocornut/imgui](https://github.com/ocornut/imgui).
