# Документация ProjectV

Структурированная документация по библиотекам стека C++/Vulkan для разработки воксельного движка.

---

## 🚀 Быстрый старт

### Ключевые технологии

1. **SDL** — создать окно.
2. **Vulkan** — нарисовать треугольник.
3. **ImGui** — добавить интерфейс.
4. **flecs** — добавить игровую логику.
